
# ⭐ Wheather App ⭐

Wheather App (web version) 

It shows Temperature,Windspeed,Humidity and Clouds.

You can search for your City.

It is build with realtime Wheather Api using JS.




# Tech Stack

HTML

CSS

Javascript

Wheater API
